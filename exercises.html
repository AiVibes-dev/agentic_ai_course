Practical Exercises and Projects

Hands-on activities to reinforce theoretical concepts and build real-world skills

Learning Through Practice

This course emphasizes hands-on learning through practical exercises and projects. Each module includes guided exercises to reinforce theoretical concepts, as well as more substantial implementation projects to build real components of agentic AI systems.

The exercises progress in complexity throughout the course, culminating in a comprehensive capstone project where you'll design and implement a complete agentic AI system for a real-world application.

Module 1: Foundations of Agentic AI Systems

Exercise 1.1: Agent Capability Analysis

Objective: Analyze existing AI agents to identify their capabilities and limitations.

Description: Select three different commercial AI agents and interact with each through a series of structured tasks to document their capabilities and limitations.

View Exercise

Exercise 1.2: Ethical Boundary Exploration

Objective: Investigate how different AI agents handle ethical dilemmas and boundary cases.

Description: Design a set of prompts that explore ethical boundaries and test these prompts across multiple AI agents to analyze their approaches to ethical guardrails.

View Exercise

Project 1: Agent Capability Prototype

Objective: Build a simple prototype demonstrating one core capability of agentic AI.

Description: Select one core capability (basic planning, short-term memory, simple tool use, or reflection/self-improvement) and implement a minimal prototype demonstrating this capability.

View Project

Module 2: Architectural Components of Agentic AI Systems

Exercise 2.1: Component Design Workshop

Objective: Design the architecture for each major component of an agentic AI system.

Description: For each core component (perception, cognition, action, memory), define responsibilities, specify inputs/outputs, identify subcomponents, and design interfaces.

View Exercise

Exercise 2.2: Memory System Implementation

Objective: Implement a basic memory system for an agentic AI.

Description: Design and implement a memory system with at least two types of memory (short-term/working memory and long-term/episodic memory).

View Exercise

Project 2: End-to-End Component Integration

Objective: Build a simple end-to-end agent by integrating basic versions of all core components.

Description: Implement simplified versions of perception, cognitive, action, and memory modules, and create a simple agent loop that processes inputs through all components.

View Project
